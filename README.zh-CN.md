### 提示

> 强烈推荐 [vue-demi](https://github.com/vueuse/vue-demi) (一款兼容 vue2 和 vue3 的包)开发项目

> 本组件库示例代码均基于 vue-demi compositional api 演示, options api 方式需自行修改部分代码。

### 特性

- 🚀 兼容 vue2 和 vue3
- 🚀 快速开发移动端业务功能
- 💪 基于 vant 等优秀的库
- 💻 所有组件均自适应屏幕
- 💚 封装兼容微信公众号组件
